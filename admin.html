  Perubahan di sini tersimpan hanya di browser ini (localStorage), sehingga langsung terlihat saat kamu membuka website dari komputer ini, tapi belum terlihat oleh pengunjung lain. Untuk menjadikannya permanen, buka tab Data & Backup, unduh JSON, lalu salin isinya ke js/data.js dan push ke GitHub.

Export & Import Data

Gunakan Export untuk mengunduh seluruh perubahan Galeri & Berita di browser ini sebagai file JSON — simpan sebagai cadangan, atau salin isinya ke js/data.js agar permanen untuk semua pengunjung. Gunakan Import untuk memuat kembali file cadangan (misalnya di komputer/browser lain).

Data Pendaftaran & Donasi

Data formulir Pendaftaran Santri dan Konfirmasi Donasi dari pengunjung tersimpan otomatis ke Google Sheets melalui backend Apps Script (Code.gs) — bukan di browser ini. Buka Google Sheets yang terhubung ke deployment Code.gs kamu untuk melihat, mengekspor ke Excel/CSV, atau mengunduhnya ke komputer kapan saja. Lihat README.md bagian "Menghubungkan Code.gs" untuk link Sheet kamu.
